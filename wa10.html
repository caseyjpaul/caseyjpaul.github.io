Adding Slide Numbers

Casey Paul

Why do we need slide numbers?

  • Slide numbers are useful because then the presenter knows where they are at in the presentation.
  • When getting feedback on a slidedeck, slide numbers make it easier to reference a specific slide.

What is the first thing I have to do?

To display slide numbers, all you have to do is set the slideNumber configuration option to true!


					Reveal.initialize({
						slideNumber: true
						});

Note: This code will need to be added to the bottom of your file.
